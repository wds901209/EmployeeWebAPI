# EmployeeWebAPI  

## 專案簡介  
**EmployeeWebAPI** 是一個基於 .NET 技術構建的 Web API，提供員工資料的 CRUD（建立、讀取、更新、刪除）操作。此專案專注於效能與安全性，並加入了驗證功能和非同步操作優化，以確保操作流暢且安全。  

---

## 功能  
- **CRUD 操作**:  
  - 新增員工資料  
  - 查詢員工資料  
  - 更新員工資料  
  - 刪除員工資料  
- **驗證功能**: 確保請求符合安全性規範。  
- **非同步操作**: 使用 `async/await` 提升效能，減少阻塞問題。  

---

## Tech Stack  
- **後端框架**: .NET  
- **資料庫**: MSSQL (使用 Entity Framework 管理資料庫操作)  
- **文件化工具**: Swagger (提供自動生成的 API 文件)
